# 🇩🇪 Deutschland Tour

**Self-guided audio walking tours through Germany's most beautiful cities.**

Explore Berlin, Munich, and Hamburg with immersive GPS-triggered audio guides. No app installation required — just open the website and start walking.

## ✨ Features

- **GPS-Activated Audio** — Stories play automatically as you approach landmarks
- **Multi-Language** — Tours available in German and English
- **Interactive Map** — Real-time position tracking via OpenStreetMap
- **No Account Required** — Start immediately, no signup
- **Pay-What-You-Want** — Berlin free; other cities as single/bundle purchases

## 🏙️ Available Tours

| City | Route | Distance | Duration |
|------|-------|----------|----------|
| Berlin | Historical Walk | 4.2 km | ~120 min |
| Munich | Old Town & Culture | 3.1 km | ~90 min |
| Hamburg | Harbor & History | 3.5 km | ~100 min |

## 🛠️ Tech Stack

```
Frontend → React 19, TypeScript, Vite, Leaflet
Backend  → Express, TypeScript, SQLite
Payments → Stripe (demo mode)
```

## 🚀 Quick Start

```bash
cd client && npm install && npm run dev   # → localhost:5173
cd server && npm install && npm run dev   # → localhost:3001
```
